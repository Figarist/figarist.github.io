## 2026-03-06 - Adding ARIA labels to dynamically generated copy buttons
**Learning:** When elements like a 'Copy' button are created dynamically in Vanilla JS and appended to code blocks, they often lack sufficient context for screen readers if they only contain the text 'Copy', especially when multiple such buttons exist on a page. Adding a specific aria-label like 'Copy Code' during the element creation process significantly improves accessibility without requiring custom CSS or breaking the zero-bloat philosophy.
**Action:** Always inspect JavaScript files that generate DOM elements dynamically (like document.createElement) to ensure they include necessary ARIA attributes (e.g., aria-label, role) alongside text content, ensuring consistent accessibility across both static HTML and dynamic UI components.
